# EpiGimp

Un éditeur d'images simplifié créé avec React et Vite.

## Installation

```bash
npm install
```

## Lancement du projet

```bash
npm run dev
```

## Fonctionnalités (Milestone 1)

✅ Layout complet avec Header, Sidebar et Workspace
✅ Routing avec React Router (Home et Editor)
✅ Import d'images (PNG, JPG, JPEG)
✅ Affichage des images avec zoom (+/-)
✅ Ajustement automatique de l'image
✅ Gestion des erreurs (format non supporté)
